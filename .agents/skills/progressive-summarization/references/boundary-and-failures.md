# 边界与失败模式

## 不要在以下情况使用此 skill

- 初次阅读时就想做到完美标注——这会导致"过度标记"（什么都觉得重要），反而让标注失效
- 没有明确输出目标时批量提炼所有笔记——这是"亏本生意"，投入时间与回报不成正比
- 需要完整保留原文精确性的法律文件、学术引文——这些场景下删减提炼有风险

## 作者在书中警告的失败模式

- **过度标记重点**：将80%以上的内容加粗或高亮，标注后和原文几乎一样长。根源是缺乏取舍的勇气——"如果什么都想要，最后就什么都得不到。"
- **漫无目的的提炼**：花一整个周末"整理笔记"但没有产出任何作品。提炼必须在"准备创作时"进行。
- **故作高深**：用理性分析每一个标注决策，而非听从直觉。应该追随内心的悸动——"哪些文字让你目不转睛、心跳加速？"

## 作者的盲点 / 时代局限

- 假设用户以文字型内容为主要信息消费形式，对播客、视频等非文字内容的提炼支持有限
- "10-20%"递减规则缺乏严格的实证依据，更多是经验法则

## 容易混淆的邻近方法论

- 传统的"画重点"学习法：一次性标注、服务于考试记忆，而非分层的、服务于长期复用。渐进式归纳法的关键差异在于"跨时间的层层叠加"和"按需深入"。
- SQ3R阅读法（浏览-提问-阅读-复述-复习）：面向理解性阅读，渐进式归纳法面向笔记的长期价值提升。
