# 书中案例 (Past Application)

## 案例 1：作者的情感支持文件夹

- **问题**：作者有时会觉得自己的工作不重要、做得不够好，产生自我怀疑。
- **方法论的使用**：他将多年来收集的客户反馈（启发性——证明工作的价值；个性——记录了真实的情感互动）保存在一个专门的文件夹中。
- **结果**："任何时候，每当我觉得自己的工作并不重要，或者做得不够好时，我便打开这个文件夹，如此便能让想法得以改观。"

## 案例 2：泰勒·斯威夫特的歌词笔记本

- **问题**：流行歌手泰勒·斯威夫特需要从日常生活的点滴感悟中提炼出打动人心的歌词。
- **方法论的使用**：她将一闪而过的歌词灵感、一段旋律、一句对白立即记录在手机笔记中。创作歌曲《空位》时，她将多年前储存的各种"金句"糅合在一起——"原来所有的爱情都只有两个归宿——要么天长地久，要么化为乌有"是她某天突然想到的；"亲爱的，我就是一出装扮成美梦的噩梦"是后来冒出的另一句。
- **结果**：《空位》成为热门金曲。她的9张专辑全球销量累计超过两亿张。"创作歌曲不是一项孤立的、只能在特定时间和地点从事的活动——它是我整个思维活动的衍生产品。"

## 案例 3：杰里·森菲尔德的黄色便签

- **问题**：喜剧大师杰里·森菲尔德需要持续产出有趣的段子。
- **方法论的使用**：每想到一个有趣的梗——无论是舞台表演时激发的灵感、闲聊时获得的启发，还是在黄色便签纸上创作时的输出——他都保存在一个老式折叠文件夹中。
- **结果**：当别人对他的记录习惯表示惊讶时，他回应："我实在不明白这有什么好惊讶的，我也实在不明白除此之外还有什么好记录的。难道还有什么能比这些更有价值么？"
