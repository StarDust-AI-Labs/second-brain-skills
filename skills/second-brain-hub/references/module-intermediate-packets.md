# 半熟素材

输入：`creation_topic`、`search_results`。输出：`matched_notes`、`usable_packets`、`material_gaps`。

将已有成果识别为可复用创作单元：干货笔记、落选素材、过程文件、最终成果、可合法复用的外部素材。

执行：

1. 根据主题筛选相关笔记和历史产物。
2. 为每项标注可直接复用、需加工或只作参考。
3. 形成最小素材包，不要求资料齐全才开始。
4. 明确缺失信息，交给创作工作流作为约束。

项目回顾和收尾时也可调用本模块，把被当前项目淘汰但未来有价值的内容转成素材包。一次性任务、机密材料和仍高速变化的原型不强制资产化。
