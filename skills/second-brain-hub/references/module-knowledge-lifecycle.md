# 知识生命周期

输入：`review_scope`、`active_projects`、`recent_notes`。输出：`review_scope`、`review_findings`、`next_priorities`。

周回顾：

1. 清理收件箱和未分类项。
2. 检查活跃项目与近期日历。
3. 识别停滞项目、下一步和下周优先事项。

月回顾：

1. 评估目标与活跃项目。
2. 检查领域是否失衡。
3. 调整优先级并归档失效内容。

项目收尾：

1. 标记结果和经验。
2. 将项目移入归档。
3. 调用半熟素材模块回收可复用产物。

无法确认回顾范围时，只报告可用数据，不创建回顾笔记。
